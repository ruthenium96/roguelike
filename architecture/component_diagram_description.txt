Приложение состоит из следующих основных компонентов:
- GameEngine отвечает за игровую логику и ход игры. Внутри есть компоненты WorldState хранящий состояние игровых объектов, а также компонент-генератор данных состояний (в том числе карты).
- UI отвечает за генерацию проекции игрового состояния и прочих элементов (e.g. игровое меню) для передачи ее в UI Painter, который накладывает изображения (GameSkin) на объекты и генерирует картинку для отображения на дисплее.
- Ввод осуществляется через InputManager, который проксирует клавиатуру или джойстик.
- Связь между компонентами GameEngine, InputManager и UI осуществляет класс Controller, который пересылает сообщения с действиями между ними.
