## Зависимости:

protobuf, googletest (optional)

## Сборка

В корневой папке проекта:
```shell
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск игры

В папке `build`:
```shell
./src/main
```

## Управление
Q - выход из игры.

Есть два режима: режим мира и режим инвентаря/экипировки. Переключение между режимами осуществляется посредством R.

В режиме мира: WASD - движение по карте, E - взаимодействие.

В режиме инвентаря/экипировки: WS - движение по инвентарю, AD - движение по экипировке, E - снять/надеть вещь, F - выбросить вещь.

## Запуск тестов:

В папке `build`:
```shell
./tests/unit/unit_tests
```
