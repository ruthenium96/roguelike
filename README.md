## Сборка

В корневой папке проекта:
```shell
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск

В папке `build`:
```shell
./src/main
```

## Управление

WASD - движение по карте, E - взаимодействие, Q - выход из игры.
